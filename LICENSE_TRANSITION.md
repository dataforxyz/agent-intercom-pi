Public License v3.0 or later (`AGPL-3.0-or-later`) on 2026-07-14.

## Boundary

- Final MIT commit: [`6fc13f7`](https://github.com/dataforxyz/agent-intercom-pi/commit/6fc13f7)
- Final MIT tag: [`mit-final`](https://github.com/dataforxyz/agent-intercom-pi/tree/mit-final)
- First AGPL commit: [`696a2f3`](https://github.com/dataforxyz/agent-intercom-pi/commit/696a2f3)
- First AGPL tag: [`agpl-transition`](https://github.com/dataforxyz/agent-intercom-pi/tree/agpl-transition)
- First versioned AGPL release: `v0.8.0`

The transition is prospective. It does not revoke or alter rights granted for copies
already received under MIT. Current and future versions derived from the AGPL side of
the boundary are distributed under `AGPL-3.0-or-later`.

The npm package named `pi-intercom` was published by the original upstream maintainer.
Those npm releases remain under their published MIT terms; this repository's AGPL
release is distributed through GitHub and Pi's Git package installer.
